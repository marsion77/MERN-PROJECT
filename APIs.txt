# Users APIs
POST http://localhost:9080/users/adduser          -> Add User
POST http://localhost:9080/users/loginUser       -> Login User
POST http://localhost:9080/users/login           -> Login OTP Generating (For Forgot Password)
POST http://localhost:9080/users/loginverify     -> Login Verification

# Category APIs
POST http://localhost:9080/category/createCategory -> Create Category

# Menu APIs
POST http://localhost:9080/menu/createMenu         -> Create Menu

# Order APIs
POST http://localhost:9080/order/addorder          -> Add Order

# Quantity APIs
POST http://localhost:9080/quantity/addQuantity    -> Add Quantity

# Order APIs
 http://localhost:9080/order/addorder    -> Add Order
